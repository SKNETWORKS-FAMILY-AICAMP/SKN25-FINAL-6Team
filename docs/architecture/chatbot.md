# Chatbot Architecture

## 역할

실시간 사용자 문의를 받아 분류하고, 가능한 경우 자동 응답 초안을 생성하는 흐름을 담당합니다.

## 예상 처리 단계

1. 사용자 메시지 수신
2. 기본 유해성/비정상 입력 점검
3. 티켓 저장
4. 문맥 조회
5. 라우터 판단
6. FAQ / 결제 / 버그 / 보이스 성격 노드 분기
7. 응답 초안 생성
8. Safety 검사
9. 자동 응답 또는 운영자 검토 전환

## 예정 구성 위치

- `apps/chatbot/`
- `packages/llm/`
- `packages/rag/`
- `packages/tools/`
- `packages/safety/`

## 문서 범위

이 문서는 추후 다음 내용을 상세화하기 위한 자리입니다.

- 상태 객체 정의
- 노드 책임 분리
- 라우터 출력 규칙
- 자동 응답 허용 조건
